# Assistente de Codigo RC / PO

Aplicacao web com fluxograma interativo para guiar o usuario ate o codigo correto de Requisicao (RC) e Pedido de Compras (PO).

## Como funciona

1. O usuario responde perguntas sequenciais.
2. A arvore de decisao calcula o caminho.
3. O sistema retorna o codigo final de RC/PO.

## Publicacao no Netlify

1. Suba este repositorio no GitHub.
2. No Netlify, clique em **Add new site > Import an existing project**.
3. Selecione o repositorio.
4. Build command: deixe vazio.
5. Publish directory: `.`.
6. Clique em **Deploy site**.
